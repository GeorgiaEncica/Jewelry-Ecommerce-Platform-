# Jewelry-Ecommerce-Platform-
Developing a fully functional online jewelry store with product management, cart system, order processing, and admin dashboard using HTML, CSS, JavaScript, PHP, and MySQL. Included functions such as Order Tracking System, 3D product view, analytics, inventory management, and admin log in.
